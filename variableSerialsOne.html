Variable Serials One

Description

This p5.js creation creates a dynamic visual canvas containing a series of interactive lines. The canvas is divided into several rectangular units, and the interior of each unit displays rich patterns through randomly generated lines. The positions, lengths, and directions of these lines are randomly determined, resulting in new patterns every time the page is refreshed or the mouse is clicked. By utilizing the mousePressed function, viewers can regenerate the entire canvas with a simple mouse click, making the work highly interactive and unpredictable.

Reflection

This project demonstrates the potential of p5.js for creating dynamic and interactive visual art. Using random functions to generate line positions and styles is a challenge during programming because it requires maintaining a certain visual order while maintaining visual appeal. Furthermore, enabling user interaction – in this case repainting the canvas via mouse clicks – adds to the engagement and dynamism of the work. This project is an exploration of the balance between control and randomness.

Matthew's artwork takes a critical view of social, political, and cultural issues. Matthew references public issues, exploring the contradictions between humanity, nature, and science.
